# QIX
![じふぅ](https://i.gyazo.com/652aa2a38b8e72a0fb02765b73c732df.gif)

QIXをvvvvで実装しました
* 十字キーで自機移動
* スペースキーを押しながら移動で領域作成

## 中身
* Image：QIXで使用する画像
* plugin：QIX用プラグイン
* QIX.v4p：サンプルパッチ

## 動作確認
* Win 8.1
* vvvv45beta34 (64bit)
