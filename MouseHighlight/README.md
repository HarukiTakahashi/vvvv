# マウスを目立たせます
![じふぅ](http://i.gyazo.com/7e0094cd6e93fe90881d902bd4bef179.gif)

マウスカーソルの周りにリング状のハイライトを表示
* Enable をオンにすれば動くはずです．
* Timing というピンにBangをつなぐと，その値変化に反応するようになります．

## メモ
* Win 8.1
* vvvv45beta33.7(64bit)  
で動作を確認しています

## できないこと
* Spreadableな入力の受け付け
* リングの大きさ変更
* 色の変更

## 注意
* スクリーンサイズを利用しているため，ディスプレイ拡張すると上手く動きません．
* プラグインをいじったりすると，Mainというタイトルのウィンドウが増殖します．  
　（うまく消してあげるか再起動してみてください...）  
プラグインが非常に雑なので，運用は自己責任でお願いします．
